# Design Patterns Implementation Repository

## Overview

This repository explores various design patterns in the context of object-oriented programming, as part of academic coursework at the university. The focus is on understanding and applying fundamental design patterns such as **Factory Method**, **Model-View-Controller (MVC)**, **Singleton**, **Observer**, and principles of **SOLID** and **Clean Code**.

The goal is to demonstrate how these patterns enhance software design, improve code maintainability, and support robust development practices.

## Design Patterns Covered

- Factory Method
- Model-View-Controller (MVC)
- Singleton
- Observer
- SOLID Principles
- Clean Code Practices

This repository serves as a resource for learning and implementing these design concepts effectively within Java.
